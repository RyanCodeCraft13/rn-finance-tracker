# RN Finance Tracker

**RN Finance Tracker** is a modern multi-bank savings and personal finance tracker.  
It allows you to track bank balances, deposits, withdrawals, transfers, set monthly budgets, and visualize your spending through charts.  

This is a Progressive Web App (PWA) that can work offline and be installed like a mobile app.

## Features
- Multi-bank balance tracking
- PIN security lock
- Deposit, Withdraw & Transfer
- Budget tracking by category
- Monthly summary & analytics charts
- Transaction history with notes
- Backup and restore via JSON
- Dark/Light theme support
- PWA ready

## Tech Stack
- HTML5 / CSS3 / JavaScript
- Chart.js
- LocalStorage
- Progressive Web App (PWA)

## How to Use
1. Enter PIN (default: 1234)
2. Add deposits, withdrawals, or transfers
3. Track your balances, budgets, and monthly summary
4. Backup data to JSON or restore from backup
